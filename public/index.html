The Unofficial DBZ API

Let me ask you... Does a machine like yourself ever experience fear?

- Vegeta

This API provides info for DBZ characters like height & weight, dates for birth & death, short descriptions and more!

Full list of characters available on Github.

Test the API here. Enter a characters name, click the button, view character data.

Quick Docs

Use url: 

https://unofficialdbzapi.cyclic.app/api/*character*

Change *character* to search for different characters.

For a quick test of the API you can ask for individual characters: 

https://unofficialdbzapi.cyclic.app/api/krillin


Example using a fetch: 


// get character from an input
let character = document.querySelector('input').value

// plug that character into API request
fetch(`https://unofficialdbzapi.cyclic.app/api/${character}`)
  .then(res => res.json())
  .then(data => {
    // use character data
  })
