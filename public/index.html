The Unofficial DBZ API

Let me ask you... Does a machine like yourself ever experience fear?

- Vegeta

This API shows info on DBZ characters like height & weight, dates for birth & death, short descriptions and more.

Test the API here. Enter a characters name, click the button, view character data.

Quick Docs

Use url: https://unofficialdbzapi.cyclic.app/api/character

Change character to whichever character you want to view.


Example using a fetch: 

          
            fetch(`https://unofficialdbzapi.cyclic.app/api/krillin`)
              .then(res => res.json())
              .then(data => {
                // do stuff with data here
              })

No API key is required, simply plug and play!
