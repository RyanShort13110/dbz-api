Unofficial Dragon Ball Z Character API

This API shows info on DBZ characters like height & weight, dates for birth & death, power levels, short descriptions and more.

Quick Docs

url: https://nice-gray-cockroach-garb.cyclic.app/character-name

Change character-name to whichever character you want to view.

For example: https://nice-gray-cockroach-garb.cyclic.app/krillin

Test it out below

Enter a characters name, click the button, view some data. Easy peasy.
